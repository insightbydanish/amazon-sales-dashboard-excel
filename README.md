# amazon-sales-dashboard-excel

Overview:- This project focuses on analyzing Amazon sales data and building an interactive dashboard using Microsoft Excel. It covers the complete workflow from raw data to insights.

Project Structure:-- The Excel file contains 4 sheets:Raw_Data – Original dataset Clean_Data – Processed and cleaned data Pivot_Analysis – Pivot tables for insights Dashboard – Final interactive dashboard with charts and slicers

Tools & Techniques Used:- *Microsoft Excel *Data Cleaning *Pivot Tables *Charts & Visualization *Slicers (Interactive Filters


Key KPIs:- *Total Revenue *Total Orders *Total Quantity Sold *Average Customer Rating *Average Discount (~13%)
